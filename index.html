### Github 정적 웹서비스를 위한 최신 디자인 트렌드가 적용된 반응형(Responsive) 웹 블로그 홈페이지 코드 만들기 - 전체 구성은 Fully Styled Website로 생성하세요. - 홈, 기술, 디자인, 소개에 맞는 화면을 만들고, 각 버튼을 누르면 해당 화면으로 이동합니다. - 첨부된 item.csv 파일 내용대로 세부 화면을 구성합니다. ### 기술 스택 및 파일 구성: - index.html (HTML5): 웹 페이지의 구조와 콘텐츠를 정의합니다. - style.css (CSS3): 모든 화면 크기에 대응하는 반응형 웹 디자인을 구현하며, 미니멀하고 깔끔한 최신 웹 디자인을 반영합니다. - script.js (JavaScript): **DOM 조작(Manipulation)**을 통한 기본적인 UI 상호작용(예: 모바일 메뉴 토글, 다크 모드 스위치)을 구현합니다. ### 주요 구성 요소 및 포함 사항: - 헤더: 로고(사이트 제목), GNB(Global Navigation Bar) 포함. - 메인 섹션: 최신 포스트 1개 (Featured Post)를 강조하고, 테스트 이미지용 플레이스홀더(Placeholder) (예: )를 사용합니다. - 포스트 그리드: 3열 구조의 최근 포스트 목록 6개를 배치하고, 각 포스트 카드에는 title, content, date, 테스트 이미지를 포함합니다. - 푸터: 저작권 정보, 소셜 미디어 링크, 간단한 내비게이션을 포함합니다. - 반응형 설계: CSS에서 **미디어 쿼리(Media Queries)**를 사용하여 데스크톱, 태블릿, 모바일 화면에서 모두 최적화된 레이아웃을 제공해야 합니다. - 배포 환경: 코드는 별도의 빌드 과정 없이 Github Pages에서 즉시 호스팅될 수 있는 순수 정적 파일로 구성되어야 합니다. ### 산출물: - SPA(Single Page Application) 형태로 출력하세요. 코드 주석을 사용하여 주요 섹션과 반응형 디자인의 **브레이크포인트(Breakpoints)**를 명확히 표시해 주세요. ### 참고사항: - 이 블로그의 사용자는 ‘Dong Gue’ 입니다. github 사이트는 'https://github.com/mskdia' 이고, Email 주소는 'mskdia@gmail.com' 입니다. - Dong Gue 이미지는 image/portfolio.jpg 를 사용합니다.
